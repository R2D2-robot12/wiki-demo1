#  项目规范

## 技术栈
- Vite + React + TypeScript
- 使用 react-router-dom v6
- 样式用 Tailwind CSS
- 动画用GSAP, Framer Motion, Three.js(已经下载好)
- 无需阅读node_modules文件夹

## 代码风格
- 组件用函数式写法
- 文件命名用 PascalCase
- 优先用 const 箭头函数

## 目录结构
- 页面放在 src/pages/
- 公共组件放在 src/components/

## 设计规格

desktop 1440px

# 具体组件

## navbar

### 尺寸

- **高度**：72px
- **左右 Padding**：80px（与页面边距一致）
- **字号**：20px，Inter，字重 700
- **字体 颜色**：`#3dffa0`

### 导航项间距

- 各链接之间间距：40px

## Hero Section

### 布局

- **高度**：100vh（全屏）
- **视频**：`object-fit: cover`，铺满全屏
- **蒙版**：`rgba(8, 15, 10, 0.55)`，绝对定位覆盖视频
- **文字容器**：绝对定位，距底部 18vh，左边距 80px

| 元素              | 内容示例             | 字号       | 字重 | 颜色      | 字间距  |
| ----------------- | -------------------- | ---------- | ---- | --------- | ------- |
|                   |                      |            |      |           |         |
| 主标题行1         | `Plants SOS`         | 96px，Syne | 700  | `#f0f5f1` | -0.03em |
| 主标题行2（高亮） | `write the subtitle` | 96px，Syne | 700  | `#3dffa0` | -0.03em |

## HeroDescription Section

### 参考视觉

- 黑色背景上大号白色粗体标题
- 关键词用亮绿色（#3dffa0）inline 高亮
- 整体风格：暗色、高对比、克制

### 触发条件
- 滚动进入视口时触发文字动画
- 使用 Intersection Observer，threshold: 0.3
- 禁止用固定 px 滚动值硬编码触发点

### 布局
- 高度：auto，最小 320px
- 背景色：#0b140d
- 内容最大宽度：760px，水平居中
- 上下 Padding：120px

### 入场动画

- 初始状态：opacity: 0，translateY(24px)
- 结束状态：opacity: 1，translateY(0)
- 持续时间：0.7s
- 缓动：cubic-bezier(0.16, 1, 0.3, 1)
- 多组文字每行 delay 错开 0.1s（第1组 0s，第2组 0.1s，第3组 0.2s）
